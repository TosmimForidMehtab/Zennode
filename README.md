## Steps to run the program

-   ### Clone the github repo using:

    ```sh
    git clone "https://github.com/TosmimForidMehtab/Zennode.git"
    ```

    ### OR

    #### Download the zipped folder

-   ### Go to the project folder

    ```sh
        cd Zennode
    ```

-   ### For Javascript Code:
    ```sh
        cd Javascript
    ```
    ```node
        npm run start
    ```
-   ### For Python Code:
    ```sh
        cd Python
    ```
    ```sh
        python main.py
    ```
    #### _For linux based OS_
    ```sh
        python3 main.py
    ```
